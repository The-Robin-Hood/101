005.MiniMax

  • An unbeatable Tic-Tac-Toe game which uses the concept of MiniMax Algorithm
  • It is a decision rule used for minimizing the possible loss for a worst case scenario.
  • The algorithm considers all possible moves that can be made by the computer and human (optimal moves).
  • Then it concludes the best move for the computer player by minimizing the maximum loss.
  • Refer to MiniMax Algorithm
